# GLFW Template
A simple template for C++ GLFW projects. Includes a make file for Linux.

## Requirements
* Make
* GCC

## Setup
### 1. Clone Repository
`git clone https://github.com/BSmith156/GLFW-Template.git`
### 2. (Linux Only) Install GLFW
`sudo apt install libglfw3-dev`
### 3. Compile Executable
`make`
